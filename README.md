# StartupUW Website

Built upon this [react-boilerplate](https://github.com/react-boilerplate/react-boilerplate)

By Aaron Chen and Connor Carlson

To begin, run: 

`npm install` then `npm start` 
    
To generate components, containers, and more:

`npm run generate`
